<b>Diversity Evaluation</b>
<p>Users can <b>Evaluate the Diversity of the Data</b> by clicking the <b>Diversity Evaluation</b> tab after clicking the <b>"Step 2 - Batch Correction"</b> tab and the first <b>Cluster Plots</b> tab.</p>
<p>To help improve calculation speed, the <b>top 2000</b> most variable features by the <b>mean absolute deviation</b> are selected by default, however, the feature number and variance measure can be easily altered by the user. These parameters can be found under the <b>Cluster Parameters</b> section of the sidepanel under the <b>"Step 2 - Batch Correction"</b> tab after selecting the <b>Diversity Evaluation</b> tab. </p>
<p>The <b>Barplot</b> generated by the <b>Diversity Evaluation</b> analysis can be colored by any feature in the meta data and can be clustered by a user specified number clusters using a variety of different methods. These parameters can be found under the <b>Cluster Parameters</b> section of the sidepanel under the <b>"Step 2 - Batch Correction"</b> tab after selecting the <b>Diversity Evaluation</b> tab. </p>
<p>Users can use the <b>Diversity Evaluation</b> to objectively determine whether the heterogeneity of the dataset <b>increases</b> when grouping by the batch and is <b>reduced</b> when grouping by the variable of interest.</p>
<p>To set up this analysis, users should first set the <b>Number of Clusters</b> to a biologically meaningful cluster number based on their dataset. Then, the user should color the dataset by the <b>batch variable</b> and determine whether heterogeneity <b>increases</b> following correction. This can be determined visually using the <b>Barplot</b> or quantitatively using the calculated heterogeneity scores provided in the data tables. The first table displays the average heterogeneity score across the entire dataset while the second table shows the heterogeneity score for each cluster. <b>BatchFLEX</b> provides multiple methods to calculate the heterogeneity score to allow the user to use their preferred method. Next, the user should color the samples by the <b>Variable of Interest</b> and determine whether the heterogeneity is <b>reduced</b>. Ideally, similar samples should cluster together based on an expected biological parameter. </p>
