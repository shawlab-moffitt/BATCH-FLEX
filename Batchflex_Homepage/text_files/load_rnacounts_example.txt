<b>Load RNASeq Counts Example</b>
<p><b>BatchFLEX</b> includes a sample RNASeq count dataset within the Shiny app, which was generated by harmonizing raw count data downloaded from Recount3. </p>
<p>Users can access the sample dataset by simply selecting the <b>Load RNASeq Counts Example</b> button from the <b>"Step 1 - Data Input"</b> tab.</p>
<p>In this dataset, <b>Study</b> is considered the batch effect and <b>Group</b> is considered the variable of interest.</p>
